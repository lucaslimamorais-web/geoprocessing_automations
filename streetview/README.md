# StreetView from UTM

Opens Google Maps Street View directly from UTM coordinates copied from ArcGIS.

## How to use
S
1. Copy the UTM coordinates from ArcGIS
2. Paste them into the app in the format: `23S 123456.789 1234567.890`
3. Click "Abrir no Google Maps"
4. Google Maps and Street View will open automatically in your browser

## Requirements

Install dependencies with:

pip install -r requirements.txt
